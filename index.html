Sejarah Curse

Curse pertama kali didirikan pada Februari 2023 dengan nama Curse Vanilla MCPE, server bertema vanilla sederhana yang berjalan hingga Mei 2023. Server ini dibuat oleh Montxyx — nama yang berasal dari nickname acak saat ia bermain Minecraft MCPE. Series kedua adalah COLTK (Curse Of The Legend Key), server bertema BHC dengan custom enchant level tinggi, berjalan dari Desember 2023 hingga Maret 2024. Di fase inilah Mont mulai membangun series Java Edition. Mont kemudian bertemu Epan (fase 2, Januari 2024), dan mulai belajar plugin serta script meski series awal belum sempurna. Perjalanan berlanjut ke server Last Warrior & LWHC, di mana Mont mengenal Sanopalz dan Zensuan. Sanopalz menjadi salah satu sosok penting yang membantu setup server. Series berikutnya adalah Cursefury Legends (CFL), awal kebangkitan besar Curse di Java Edition. Mont merekrut Rexdzio, Ahmad (PakSamsul7), serta didukung Latif. Kemudian hadir Curseborne Kingdom. Setelah 3 tahun perjalanan (2023–2026), Mont memutuskan membuat series terakhir sebagai bentuk penghormatan kepada Curse Community. Kini lahirlah server terakhir: Survival RPG dengan custom enchant luar biasa dan dungeon menantang. Mont berterima kasih kepada seluruh member dan support system yang telah menemani perjalanan ini.
